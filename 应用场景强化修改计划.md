# 应用场景强化修改计划

## 目标

把当前项目从“老照片主题包装下的课程算法工具箱”，调整为“面向家庭老照片修复与纪念化输出的完整应用”。保留现有 40 项算法和风格迁移成果，但把报告、演讲、PPT、GUI 的主线统一为：

> 导入老照片 → 识别退化问题 → 一键修复 → 针对性精修 → 划痕/模糊修复 → 艺术纪念版生成 → 导出结果

## Phase 1：应用定位与文稿结构重写

### 修改文件

- `报告.md`
- `演讲稿.md`
- `README.md`

### 修改内容

1. 去掉面向答辩展示中过度显眼的“难度系数 1.3 / 系数 +0.3 / 章节覆盖”表述，改为内化表达：完整系统 + 风格迁移进阶功能。
2. 报告新增“老照片退化问题分析”章节，建立问题—现象—处理策略表：
   - 发黄偏色 → 白平衡
   - 低对比 / 灰蒙 → CLAHE / Gamma
   - 颗粒噪声 → 中值 / NLM / 双边
   - 轻微模糊 → USM / 维纳滤波
   - 划痕折痕 → 掩膜 + inpaint
   - 纪念化需求 → Gatys 风格迁移
3. 报告结构从“功能清单优先”改成“应用流程优先”：
   - 项目概述
   - 老照片问题分析
   - 应用流程设计
   - 系统架构
   - 算法支撑
   - GUI 设计
   - 实验评估
   - 总结
4. 演讲稿 15 页按应用流程重排，不再逐页像算法清单巡礼。
5. README 首屏改为面向使用者的项目说明，保留运行命令和交付物说明。

### 验收标准

- 读报告前 2 页能明确看出“家庭老照片修复与纪念化应用”主题。
- 演讲稿前 3 页先讲用户场景、流程、系统入口，而不是功能数量。
- 搜索 `难度系数|系数 1.3|+0.3|第X章` 不再出现在面向展示的主交付物中。

## Phase 2：PPT 主线与视觉内容重构

### 修改文件

- `代码/build_slides.py`
- 生成 `演示文稿.html`

### 修改内容

将 15 页幻灯片重排为：

1. 封面：老照片修复与风格化迁移纪念
2. 应用动机：家庭老照片为什么需要修复
3. 用户流程：导入 → 修复 → 精修 → 纪念版 → 导出
4. 退化问题地图：偏色、低对比、噪声、模糊、划痕
5. 系统总览：普通修复入口 + 高级算法工具箱
6. 一键修复流水线：去噪 → 去偏色 → 增强 → 锐化
7. 去黄与提亮：白平衡 / CLAHE / Gamma
8. 去颗粒噪声：中值 / NLM / 双边
9. 划痕与模糊修复：inpaint / 维纳滤波
10. 高级算法工具箱：分割 / 形态学 / 频域 / 小波，压缩讲
11. 艺术纪念版生成：Gatys 原理
12. 纪念版效果：修复版 → 艺术风格版
13. 量化评估：合成老照片 PSNR / SSIM
14. 真实案例：问题标注 + 修复后改善
15. 总结与 Q&A

### 验收标准

- 幻灯片标题以“用户问题 / 应用流程”为主，不以算法章节或功能清单为主。
- “风格迁移”在展示中被解释为“艺术纪念版生成”。
- 真实老照片案例页包含“修复前问题”和“修复后改善”的文字标注。

## Phase 3：GUI 应用入口增强

### 修改文件

- `代码/gui/app.py`
- 可选：`代码/dip/pipeline.py`

### 修改内容

1. 在左侧功能面板最上方增加“老照片修复向导”分组，作为普通用户入口。
2. 向导按钮内部调用现有函数，不新增复杂算法：
   - 一键温和修复：调用 `pipeline.one_click_restore`
   - 去黄偏色：调用 `enhance.white_balance`
   - 提亮与增强：调用 `enhance.clahe`
   - 去颗粒噪声：调用 `smooth.median_blur` 或 `smooth.nlmeans`
   - 增强清晰度：调用 `sharpen.unsharp_mask`
   - 艺术纪念版：打开风格迁移面板
3. 将底部独立按钮“风格迁移 [进阶]”改名为“生成艺术纪念版”。
4. 将“划痕修复 [加载掩膜]”改为“划痕/折痕修复（选择掩膜）”，提示白色区域表示待修补区域。
5. 关于弹窗从“覆盖章节”改成“面向老照片修复与纪念化输出”。
6. 若时间允许，在 `pipeline.py` 增加 `preset` 参数：温和 / 清晰 / 去偏色 / 去噪，GUI 一键修复可选择预设。

### 验收标准

- 打开 GUI 后，左侧第一组就是应用导向的修复向导。
- 老师或普通用户不需要理解算法名，也能知道第一步该点什么。
- 高级算法分组仍保留，作为课程展示和精修工具。
