# 视频录制操作说明（照着点即可）

> 目标：录一段 70–90 秒的系统演示视频，放进演示文稿第 12 页。  
> 不需要现场理解算法，录制时按下面顺序点击即可。

---

## 一、录制前准备

### 1. 打开项目目录

在文件资源管理器打开：

```text
E:\数字图像处理\项目
```

### 2. 准备录屏工具

任选一种：

- Windows 自带 Xbox Game Bar：按 `Win + G`，点“录制”。
- OBS：录制整个屏幕或录制窗口。
- PowerPoint / 剪映 / 其他录屏工具也可以。

录制要求：

- 横屏 16:9。
- 只录屏幕，不需要录麦克风。
- 时长控制在 70–90 秒。
- 最终导出为 MP4，文件名改成：

```text
demo.mp4
```

---

## 二、启动程序

在 PowerShell 运行：

```powershell
cd E:\数字图像处理\项目
E:\miniconda3\envs\ai-service\python.exe 代码\main.py
```

窗口标题应该是：

```text
老照片修复与风格化迁移纪念
```

如果窗口太小，手动拉大一点，建议占屏幕大约 80%。录制时要能看到：

- 左侧功能按钮。
- 中间“原图”。
- 右侧“当前结果”。

---

## 三、录制用图片

首选真实老照片：

```text
E:\数字图像处理\项目\素材\真实老照片\R.jpg
```

备用真实老照片：

```text
E:\数字图像处理\项目\素材\真实老照片\R (1).jpg
```

如果不想展示真实照片，就用合成老照片：

```text
E:\数字图像处理\项目\素材\合成老照片\test_old.jpg
```

推荐：优先用 `R.jpg`，效果最像真实项目。

---

## 四、正式录制脚本（按时间顺序）

### 0–5 秒：展示程序界面

1. 开始录屏。
2. 鼠标停在程序窗口上。
3. 不要急着点，让画面停 2 秒。
4. 让观众看到左侧有很多功能组。

录制目的：证明这是一个完整 App，不是几张图片。

---

### 5–15 秒：打开老照片

1. 点击菜单栏：

```text
文件 → 打开
```

2. 在文件选择框中进入：

```text
E:\数字图像处理\项目\素材\真实老照片\
```

3. 选择：

```text
R.jpg
```

4. 点击“打开”。
5. 等待原图显示在左侧“原图”区域。

停 1 秒，让画面稳定。

---

### 15–28 秒：点击温和修复

在左侧找到分组：

```text
家庭老照片修复
```

点击按钮：

```text
温和修复
```

等待右侧“当前结果”刷新。

停 2 秒，让观众看到前后对比。

这一步展示：默认修复，照片更干净、更自然。

---

### 28–40 秒：点击泛黄褪色修复

继续在左侧“家庭老照片修复”分组里点击：

```text
泛黄褪色修复
```

等待右侧刷新。

停 2 秒。

这一步展示：减少黄化、改善旧照偏色。

如果你觉得效果不明显，也没关系，录屏里只要点到即可。

---

### 40–52 秒：点击人像清晰化

继续点击：

```text
人像清晰化
```

等待右侧刷新。

停 2 秒。

这一步展示：人物轮廓和细节增强。

如果按钮位置不在屏幕内，就在左侧面板稍微向下滚动一点。

---

### 52–65 秒：点击纪念明信片风

在左侧找到分组：

```text
纪念版生成
```

点击按钮：

```text
纪念明信片风
```

等待右侧刷新。

停 2–3 秒。

这一步很重要，视觉效果最明显。它展示“修复后可以生成纪念版”。

---

### 65–72 秒：快速展示复古相册风

继续在“纪念版生成”里点击：

```text
复古相册风
```

等待右侧刷新。

停 1–2 秒。

这一步展示：不止一个模板。

---

### 72–82 秒：展示 VGG 高级风格入口

在“纪念版生成”分组下面点击：

```text
自定义艺术风格迁移（VGG）
```

会弹出新窗口，标题是：

```text
自定义艺术风格迁移（VGG）
```

在弹窗里做这些动作：

1. 点选风格图缩略图：

```text
warm_family_memory_style
```

如果没有这个名字，就选任意一个看起来偏暖色的缩略图。

2. 把鼠标移动到参数区，让观众看到：

```text
风格强度
迭代步数
处理尺寸
开始迁移
```

3. 不要点击“开始迁移”。

原因：VGG 会运行较久，录屏没必要等。

---

### 82–90 秒：关闭 VGG 窗口并导出对比图

1. 点击 VGG 弹窗右上角关闭。
2. 回到主窗口。
3. 点击菜单栏：

```text
文件 → 导出对比图
```

4. 文件名随便填，例如：

```text
video_demo_compare.jpg
```

5. 保存到：

```text
E:\数字图像处理\项目\结果\
```

6. 看到状态栏出现“对比图已导出”即可。
7. 停 1 秒，结束录屏。

如果导出对比图这一步来不及，可以省略，直接在 VGG 弹窗后结束录屏。

---

## 五、录完后怎么放进 PPT

把录好的视频复制到：

```text
E:\数字图像处理\项目\素材\演示视频\
```

并改名为：

```text
demo.mp4
```

然后在 PowerShell 运行：

```powershell
cd E:\数字图像处理\项目
E:\miniconda3\envs\ai-service\python.exe 代码\build_slides.py
```

运行成功后，会重新生成：

```text
E:\数字图像处理\项目\演示文稿.html
```

这个 HTML 会把视频直接嵌进去，不需要再单独带 `demo.mp4`。

## 六、如果录屏时出问题

### 问题 1：真实照片打不开

改用：

```text
E:\数字图像处理\项目\素材\合成老照片\test_old.jpg
```

### 问题 2：VGG 弹窗里没有 warm_family_memory_style

随便点一个风格图缩略图即可，例如：

```text
starry_night
```

录屏只需要展示入口和参数，不需要真的运行。

### 问题 3：左侧按钮太多找不到

只需要找到这几个按钮：

```text
温和修复
泛黄褪色修复
人像清晰化
纪念明信片风
复古相册风
自定义艺术风格迁移（VGG）
```

找不到就滚动左侧功能栏。

### 问题 4：视频太长

剪掉等待和鼠标停顿，保留这些关键画面即可：

```text
打开图片 → 温和修复 → 泛黄褪色修复 → 人像清晰化 → 纪念明信片风 → VGG 弹窗 → 导出对比图
```

### 问题 5：视频太大

用剪映、格式工厂或 OBS 降低码率。目标：

```text
100 MB 以内
```

因为视频会嵌入 HTML，太大会导致 `演示文稿.html` 很大。

---

## 七、最终检查清单

录完后确认：

- [ ] 视频文件名是 `demo.mp4`。
- [ ] 视频放在 `E:\数字图像处理\项目\素材\演示视频\demo.mp4`。
- [ ] 已运行 `代码\build_slides.py`。
- [ ] 打开 `演示文稿.html`，第 12 页能看到视频。
- [ ] 点击视频播放按钮，能正常播放。
- [ ] 视频时长大约 70–90 秒。
