## 一、项目概述

### 1.1 项目名称与定位

**老照片修复与风格化迁移纪念** 是一个面向家庭老照片的桌面图像处理应用。用户导入一张发黄、灰蒙、有颗粒噪声、轻微模糊或带划痕的老照片后，可以先用“一键修复”得到自然的基础修复结果，再根据具体问题继续精修，最后可生成艺术纪念版，用于相册封面、纪念海报或礼物卡片。

项目保留完整的经典数字图像处理工具箱，同时将主要交互流程设计为“普通用户能理解的修复向导”：

```text
导入老照片 → 判断退化问题 → 一键修复 → 针对性精修 → 划痕/模糊修复 → 艺术纪念版生成 → 导出结果
```

### 1.2 选题动机

- **真实需求明确**：家庭老照片常见褪色、发黄、划痕和扫描噪声，修复前后差异直观，适合课堂演示。
- **数字图像处理特征明显**：去噪、增强、锐化、恢复、频域分析、形态学处理都能自然服务于老照片修复。
- **进阶功能自然融入**：风格迁移不作为孤立技术展示，而是作为“修复后的艺术纪念版生成”，让项目从修复走向纪念化输出。
- **零训练、可复现**：经典修复部分使用 OpenCV / NumPy；风格化部分使用预训练 VGG19，不训练网络、不依赖数据集。

## 二、老照片退化问题分析

老照片修复不是简单堆叠滤镜，而是先判断照片问题，再选择合适处理策略。本项目把常见退化现象整理为下面的应用映射。

| 老照片问题 | 用户看到的现象 | 处理策略 | 系统功能 |
| :--- | :--- | :--- | :--- |
| 发黄偏色 | 整体偏黄、偏棕、肤色不自然 | 估计通道均值并做白平衡 | 白平衡 |
| 低对比 / 灰蒙 | 画面发灰、人物轮廓不突出 | 拉伸局部或全局灰度分布 | CLAHE、直方图均衡、Gamma |
| 颗粒噪声 | 扫描颗粒、胶片噪点、椒盐点 | 邻域统计或自相似去噪 | 中值滤波、双边滤波、NLM |
| 轻微模糊 | 边缘不清、人脸细节弱 | 细节层增强或退化模型恢复 | USM、维纳滤波 |
| 划痕 / 折痕 | 白线、裂痕、局部破损 | 掩膜标出破损区域并自动填补 | inpaint 修复 |
| 需要纪念化 | 想做相册封面、艺术海报 | 保留内容结构并迁移艺术纹理 | Gatys 风格迁移 |

这个表也是系统设计的核心依据：普通用户先按照“问题”选择入口，高级用户再进入完整算法工具箱调参。

## 三、应用流程设计

### 3.1 普通用户流程

1. **打开照片**：从本地选择 jpg/png/bmp 等格式的老照片。
2. **一键温和修复**：系统按默认流水线自动完成去噪、去偏色、增强和轻微锐化。
3. **针对性精修**：如果仍有明显偏色、噪声、模糊或划痕，可使用修复向导中的单项功能继续处理。
4. **艺术纪念版生成**：选择《星空》《神奈川冲浪里》或自定义风格图，生成艺术化纪念图。
5. **保存 / 导出对比图**：保存修复结果，或导出“原图 + 修复图”的并排对比图用于展示。

### 3.2 高级展示流程

为了满足课堂演示和算法理解，系统同时保留完整算法工具箱，包括基础操作、增强、去噪、锐化、分割、形态学、图像恢复、频域处理、小波去噪等模块。它们不是普通用户的主入口，而是用于：

- 展示各类经典算法的效果差异；
- 调整参数观察图像变化；
- 对一键修复结果继续细调；
- 支撑报告中的定量实验和对比图生成。

## 五、核心处理策略

### 5.1 一键修复流水线

一键修复面向普通用户，目标不是让每个指标都最高，而是在多数老照片上得到自然、稳定、不过冲的初始结果。

```text
可选划痕修补 → 去噪 → 白平衡 → 对比度增强 → 轻微锐化
```

- **先去噪**：避免后续增强和锐化把噪点一起放大。
- **再去偏色**：用 Gray World 思想校正发黄、偏棕等通道失衡问题。
- **再增强**：CLAHE 拉开局部对比度，让人物轮廓和背景层次更清楚。
- **最后锐化**：USM 补偿去噪带来的轻微模糊，但使用较温和强度，避免边缘过冲。

### 5.2 去黄、提亮与对比度增强

老照片常见偏暗和灰蒙感。系统提供：

- **白平衡**：按各通道均值进行校正，使色彩回到更自然的状态。
- **直方图均衡化**：重新分配灰度分布，扩大动态范围。
- **CLAHE**：分块做自适应均衡，并通过 `clipLimit` 限制过度增强。
- **Gamma 变换**：针对暗部提亮或亮部压暗，适合手动微调。

### 5.3 颗粒噪声处理

不同噪声对应不同方法：

- **中值滤波**：对椒盐点和孤立颗粒特别有效。
- **高斯滤波**：适合平滑轻微高斯噪声，但会模糊边缘。
- **双边滤波**：同时考虑空间距离和颜色差异，能在平滑时保留边缘。
- **非局部均值**：利用图像自相似性搜索相似 patch，主观效果更自然，但速度较慢。

### 5.4 划痕、折痕与模糊修复

划痕修复采用 `cv2.inpaint`：用户提供一张白色区域表示待修补位置的掩膜，系统根据周围像素自动填补破损区域。它适合演示老照片中的白线、折痕、扫描破损等局部问题。

模糊修复采用退化模型：

$$g(x,y)=f(x,y)*h(x,y)+n(x,y)$$

其中 $h$ 表示运动模糊核，$n$ 表示噪声。逆滤波直接除以 $H(u,v)$，在 $H$ 接近零时会放大噪声；维纳滤波加入噪功比参数 $K$，在恢复细节和抑制噪声之间折中。

### 5.5 艺术纪念版生成

风格迁移采用 Gatys 方法：

1. 使用 ImageNet 预训练 VGG19 作为特征提取器，不训练网络。
2. 内容图由深层特征表示，用来保留人物、姿态和构图。
3. 风格图由多层特征的 Gram 矩阵表示，用来表达笔触、纹理和色彩相关性。
4. 从内容图出发优化输出图像素，最小化内容损失与风格损失。

应用层面上，修复版用于保留真实记忆，艺术纪念版用于相册封面、纪念海报或节日礼物。

## 七、实验评估

### 7.1 评估方法

采用“合成退化 + 真实老照片”双轨评估：

- **合成退化**：对干净原图叠加高斯噪声、低对比、偏色、轻微模糊和随机划痕。因为有干净原图，所以可以计算 PSNR 和 SSIM。
- **真实老照片**：使用实际家庭扫描件。真实图没有标准干净原图，因此用主观维度评价偏色、对比度、噪声、细节和自然度。

### 7.2 量化结果（合成老照片）

| 方法 | PSNR (dB) | SSIM |
| :--- | :--- | :--- |
| 退化老照片（基线） | 16.15 | 0.4007 |
| 中值滤波 k=3 | 17.37 | 0.5743 |
| 中值滤波 k=5 | 17.02 | 0.4949 |
| 高斯滤波 k=5 | 16.99 | 0.5033 |
| 双边滤波 | 16.04 | 0.3622 |
| 非局部均值 h=8 | 16.24 | 0.4018 |
| 非局部均值 h=12 | 16.33 | 0.3887 |
| CLAHE | 15.84 | 0.4144 |
| 直方图均衡 | 17.36 | 0.4538 |
| 白平衡+CLAHE | 16.11 | 0.4163 |
| 小波去噪 t=20 | 16.48 | 0.4040 |
| 一键修复（默认） | 16.02 | 0.4231 |
| **一键修复（中值）** | **18.06** | **0.5964** |
| 一键修复（双边） | 16.65 | 0.4372 |

**分析**：

- 中值滤波版一键修复在合成退化样张上表现最好，PSNR 从 16.15 提升至 18.06，SSIM 从 0.4007 提升至 0.5964。
- 默认一键修复使用更温和的 NLM / CLAHE / USM 组合，主观上更自然，但在这组合成退化下不一定取得最高 PSNR。
- 增强类操作会重新映射像素值，可能降低 PSNR，但能明显改善灰蒙、偏暗等主观问题。
- 因此本项目同时看量化指标和真实照片视觉效果，避免只用 PSNR 判断修复质量。

### 7.3 真实老照片主观评价

| 评价项 | 修复前 | 修复后 | 说明 |
| :--- | :--- | :--- | :--- |
| 偏色 | 整体发黄、色彩不均 | 色彩更中性 | 白平衡有效 |
| 对比度 | 画面灰蒙、层次弱 | 人物轮廓更清楚 | CLAHE 改善局部对比 |
| 噪声 | 背景和暗部颗粒明显 | 颗粒减少 | 去噪有效，但保留一定老照片质感 |
| 细节 | 边缘略模糊 | 轮廓更清晰 | USM 提升可见细节 |
| 自然度 | 原始老旧感强 | 更适合观看与保存 | 强度需要控制，避免过度锐化 |

### 7.4 艺术纪念版效果

修复后老照片与《星空》等风格图结合后，人物结构和空间关系仍保留，色彩和笔触转为艺术风格。该功能的价值不在“修复真实感”，而在“纪念化再创作”：保留一张真实修复版，再生成一张艺术版用于展示。

## 八、技术难点与解决方案

### 8.1 Windows 中文路径问题

**问题**：OpenCV 的 `cv2.imread` / `cv2.imwrite` 在 Windows 中文路径下会读取失败。

**解决方案**：封装 `utils/imgio.py`，读取用 `numpy.fromfile` + `cv2.imdecode`，写入用 `cv2.imencode` + `ndarray.tofile`。全项目统一调用该模块。

### 8.2 维纳滤波参数调优

**问题**：维纳滤波的噪功比参数 $K$ 需要根据噪声水平调整。$K$ 太大图像过度平滑，$K$ 太小则噪声被放大。

**解决方案**：GUI 中提供 K 值滑块，范围 0.001–0.1，默认 0.01。配合退化模拟对比不同 K 值的效果。

### 8.3 风格迁移显存管理

**问题**：VGG19 + L-BFGS 风格迁移较耗显存，大尺寸图像容易 OOM。

**解决方案**：限制处理长边 ≤ 720px，并在 GUI 中开放尺寸滑块。512px 下在 RTX 4050 上约 21 秒可得到结果。

### 8.4 tkinter 后台线程

**问题**：风格迁移耗时较长，如果直接在主线程运行会导致界面无响应。

**解决方案**：用后台线程执行迁移，通过 progress 回调和 `after()` 在主线程更新进度条，完成后再回写当前结果。

## 十、总结

本项目围绕“老照片修复与风格化迁移纪念”构建了一个可交互桌面应用。它不是单纯展示算法清单，而是从真实照片问题出发，把偏色校正、对比度增强、噪声抑制、细节恢复、划痕修补和艺术纪念版生成串成完整流程。

项目的核心价值有三点：

1. **应用完整**：普通用户可以按修复向导完成一张老照片从导入到导出的处理流程。
2. **算法扎实**：高级工具箱保留经典数字图像处理算法，便于参数实验和课堂展示。
3. **纪念化输出**：Gatys 风格迁移让修复后的照片不仅能保存真实记忆，也能生成个性化艺术纪念图。

通过合成退化实验，一键修复（中值版）将 PSNR 从 16.15 提升至 18.06，SSIM 从 0.4007 提升至 0.5964。真实老照片案例也显示出偏色校正、对比度提升和噪声减少等直观改善。

项目代码已开源：https://github.com/tugumineko/dip-course-project
