## 一、项目概述

### 1.1 项目目标

本项目实现了一套面向**家庭老照片修复与纪念化生成**的桌面应用。用户导入一张家庭旧照后，可以根据照片类型选择合适的修复模式，例如温和修复、泛黄褪色修复、黑白旧照增强、扫描颗粒修复、人像清晰化、折痕划痕辅助修复。修复后，用户还可以生成复古相册、暖色胶片、纪念明信片、淡彩手绘等纪念版输出。

项目希望达到两个目的：

1. **修复目的**：改善家庭老照片常见的偏黄、褪色、灰蒙、颗粒、模糊、折痕和划痕问题，使照片更适合保存和观看。
2. **纪念目的**：在保留原照片情感内容的基础上，生成适合相册、展示、礼物卡片的纪念版图像。

### 1.2 系统功能总览

系统由四类模块组成：

| 模块 | 面向需求 | 主要功能 | 输出 |
| :--- | :--- | :--- | :--- |
| 家庭老照片修复 | 解决照片质量退化 | 温和修复、黄化修复、黑白增强、颗粒修复、人像清晰化、划痕修复 | 更清晰自然的修复图 |
| 纪念版生成 | 生成适合展示的纪念图 | 复古相册、暖色胶片、纪念明信片、淡彩手绘 | 可保存/展示的纪念版 |
| VGG 自定义艺术风格 | 满足个性化高级风格需求 | 预训练 VGG19 + Gatys 风格迁移 | 用户自定义艺术风格图 |
| 高级算法功能 | 支撑实验、调参与课程展示 | 增强、去噪、锐化、恢复、频域、小波、分割、形态学 | 单算法处理结果 |

## 二、需求分析

### 2.1 家庭老照片的修复需求

家庭老照片通常有几个特点：

- **照片内容有情感价值**：人物、家庭场景和年代感需要保留，不能过度滤镜化。
- **退化类型混合**：同一张照片可能同时发黄、褪色、低对比、有颗粒、略模糊。
- **修复结果要自然**：目标不是把老照片变成手机新照片，而是让它更清楚、更干净，同时保留旧照质感。
- **操作应该直接**：用户更关心“这张照片泛黄了”“这张照片颗粒多”“这张人像不清”，而不是具体算法名称。

因此，系统将常见需求整理为六个修复模式。

| 需求 | 用户看到的问题 | 系统提供的模式 | 设计目标 |
| :--- | :--- | :--- | :--- |
| 通用改善 | 轻微发黄、灰蒙、颗粒 | 温和修复 | 稳定、自然、不过度 |
| 色彩老化 | 偏黄、偏棕、褪色 | 泛黄褪色修复 | 去除过量黄化，保留暖色记忆感 |
| 黑白旧照 | 灰度层次弱、边缘不清 | 黑白旧照增强 | 提高明暗层次和相纸质感 |
| 扫描噪声 | 背景颗粒、暗部脏点 | 扫描颗粒修复 | 去噪但保护人物轮廓 |
| 人像不清 | 脸部和轮廓略模糊 | 人像清晰化 | 增强中等边缘，抑制噪声放大 |
| 局部破损 | 折痕、划痕、白线 | 折痕划痕辅助修复 | 自动辅助掩膜 + 用户掩膜 + inpaint |

### 2.2 家庭纪念版的生成需求

修复后的照片除了保存真实版本，还常用于纪念展示。家庭照片的纪念风格应尽量温和、怀旧、贴近相册，而不是强烈改变主体内容。因此系统提供四种传统图像处理模板：

| 需求 | 模板 | 设计目的 |
| :--- | :--- | :--- |
| 家庭相册保存 | 复古相册风 | 低饱和、暖色、暗角、米色边框，突出旧相册质感 |
| 怀旧生活照 | 暖色胶片风 | 用胶片曲线、颗粒和暖高光表现年代感 |
| 课堂展示 / 礼物卡片 | 纪念明信片风 | 将修复照片排版到米色画布，带白边和标题区域 |
| 柔和再创作 | 淡彩手绘风 | 平滑色块并叠加浅线稿，生成温和绘画感 |

### 2.3 高级自定义风格需求

部分用户可能希望上传任意风格图进行艺术化生成。因此系统保留 VGG19 Gatys 风格迁移模块，作为高级功能。它不承担默认纪念模板的职责，而是提供开放式风格扩展。

## 三、模块设计与算法目的

### 3.1 家庭老照片修复模块

代码位置：`代码/dip/family_restore.py`

该模块提供六个场景化修复函数。每个函数都由多个传统图像处理步骤组成，目的是对应一种家庭照片需求。

#### 3.1.1 温和修复

**目的**：处理大多数轻度退化的家庭旧照，作为默认推荐修复方式。

**需求来源**：很多老照片并不需要强烈修复，只需要减少灰蒙、颗粒和轻微偏色。

**处理流程**：

```text
轻度 NLM 去噪 → 黄化校正与亮度增强 → 低强度 CLAHE → 亮度通道温和 USM
```

**设计理由**：

- NLM 强度设置较低，避免把人物皮肤和衣物纹理抹平；
- CLAHE 使用低强度参数，增强层次但不过曝；
- USM 只做轻微锐化，并用噪声门限避免锐化暗部噪点。

#### 3.1.2 泛黄褪色修复

**目的**：处理整体偏黄、偏棕、褪色的彩色老照片。

**需求来源**：老相纸和保存环境容易导致色彩黄化，但完全去黄会让照片失去年代感。

**处理流程**：

```text
BGR → LAB → 估计 b 通道黄色偏移 → 限幅校正 → L 通道 CLAHE → 轻微饱和度恢复
```

**设计理由**：

- 在 LAB 空间中分离亮度和颜色，更容易控制黄色偏移；
- 只校正过量黄化，保留少量暖色；
- 增强亮度通道而不是直接增强 RGB，减少偏色风险。

#### 3.1.3 黑白旧照增强

**目的**：增强黑白旧照的层次、轮廓和纸张质感。

**需求来源**：黑白照片通常不需要色彩恢复，而是需要更清楚的灰度层次。

**处理流程**：

```text
灰度化 → 轻度去噪 → CLAHE → S 曲线增强 → 细节锐化 → 可选暖纸色
```

**设计理由**：

- CLAHE 增强局部灰度差异；
- S 曲线强化中间调，使人物轮廓更清楚；
- 暖纸色用于模拟老相纸效果，增强纪念感。

#### 3.1.4 扫描颗粒修复

**目的**：减少扫描颗粒和胶片噪声，同时保护人物边缘。

**需求来源**：扫描件常见背景颗粒和暗部脏点，直接强去噪会导致人物轮廓变糊。

**处理流程**：

```text
YCrCb 分离 → 色度通道强去噪 → 亮度通道弱去噪 → Canny 边缘权重 → 原图与去噪图融合
```

**设计理由**：

- 色度噪声可以更强处理，因为它对结构细节影响较小；
- 亮度通道保守处理，保留细节；
- 边缘区域更多保留原图，平坦区域更多使用去噪结果。

#### 3.1.5 人像清晰化

**目的**：改善家庭合影或人像照片中人物轮廓不清的问题。

**需求来源**：家庭老照片常以人物为主体，但项目不引入人脸识别模型，仍使用传统图像处理完成清晰化。

**处理流程**：

```text
亮度通道 → 双边滤波得到 base → detail = 原亮度 - base → 梯度 mask + 亮度 gate → 局部增强
```

**设计理由**：

- 只增强中等边缘，避免把噪声当细节放大；
- 暗部区域使用亮度 gate 抑制锐化；
- 双边滤波提供边缘保护的基础层。

#### 3.1.6 折痕划痕辅助修复

**目的**：修补明显细线划痕、折痕和局部破损。

**需求来源**：老照片常见白线、折痕、裂痕。完全自动识别不稳定，因此系统采用“自动辅助 + 用户掩膜”的方式。

**处理流程**：

```text
灰度图 → top-hat / black-hat 提取亮线和暗线 → 阈值分割 → 膨胀掩膜 → 合并用户掩膜 → inpaint
```

**设计理由**：

- top-hat 适合提取细亮线，black-hat 适合提取细暗线；
- 自动掩膜用于辅助，用户掩膜用于补充复杂破损；
- inpaint 根据周围像素填补掩膜区域。

## 四、纪念版生成模块

代码位置：`代码/dip/commemorative.py`

该模块使用传统图像处理方法生成纪念风格，目标是快速、稳定、贴合家庭照片语境。

### 4.1 复古相册风

**目的**：生成类似家庭旧相册中的照片效果。

**处理方式**：降低饱和度、增加暖色、添加柔和暗角和米色相纸边框。

**适用场景**：长辈旧照、家庭合影、相册保存。

### 4.2 暖色胶片风

**目的**：生成怀旧胶片质感。

**处理方式**：S 曲线增强、暖高光、轻微颗粒、轻暗角。

**适用场景**：生活照、年轻时期照片、年代感展示。

### 4.3 纪念明信片风

**目的**：生成适合展示或赠送的纪念卡片。

**处理方式**：将照片置于米色画布上，加白边、投影和标题区域。

**适用场景**：课堂展示、相册封面、节日礼物卡片。

### 4.4 淡彩手绘风

**目的**：在保留照片内容的同时生成柔和绘画感。

**处理方式**：双边滤波平滑色块，Canny 生成浅线稿，再与原图色彩融合。

**适用场景**：柔和纪念图、手绘风输出。

## 五、VGG 自定义艺术风格模块

代码位置：`代码/style/neural_style.py`

### 5.1 模块目的

该模块用于满足更自由的个性化艺术风格需求。用户可以上传任意风格图，系统使用预训练 VGG19 进行 Gatys 风格迁移。

### 5.2 方法说明

- 使用 ImageNet 预训练 VGG19 作为特征提取器；
- 内容图由深层特征图表示，用来保留人物结构和构图；
- 风格图由多层 Gram 矩阵表示，用来表达纹理、笔触和色彩相关性；
- 输出图从内容图初始化，通过 L-BFGS 优化像素，不训练神经网络。

### 5.3 家庭风格参考图

为了让 VGG 风格迁移更贴合家庭纪念主题，项目提供 `素材/风格图/warm_family_memory_style.jpg` 作为暖调家庭记忆风格参考，用于展示柔和、怀旧、偏家庭相册语境的自定义艺术风格。

## 七、实验与评估

### 7.1 展示素材

`代码/generate_gallery.py` 生成三类结果：

1. `custom_*.jpg`：六种家庭老照片修复模式的前后对比；
2. `memory_*.jpg`：四种纪念版模板效果；
3. `metrics.json` / `metrics_chart.jpg`：合成样张上的 PSNR / SSIM 指标。

### 7.2 合成样张量化结果

| 方法 | PSNR (dB) | SSIM |
| :--- | :--- | :--- |
| 退化老照片（基线） | 16.15 | 0.4007 |
| 温和修复 | 16.64 | 0.4476 |
| 泛黄褪色修复 | 16.49 | 0.4209 |
| 扫描颗粒修复 | 16.24 | 0.4170 |
| 人像清晰化 | 15.97 | 0.3987 |
| 一键修复（中值） | 18.06 | 0.5964 |

### 7.3 指标解释

合成样张有干净原图，因此可以计算 PSNR 和 SSIM。但家庭老照片修复还需要考虑主观自然度：

- 泛黄褪色修复会主动改变颜色，PSNR 不一定最高；
- 人像清晰化增强边缘，可能增加像素差异；
- 纪念版模板主动添加色调、边框和排版，不宜只用 PSNR 衡量；
- 真实家庭照片没有标准干净原图，需要看偏色、噪声、清晰度和纪念效果。

因此项目采用“合成样张量化 + 真实照片主观展示”的评估方式。

## 九、总结

本项目完成了一套家庭老照片修复与纪念化生成系统。各模块的目的如下：

- **家庭老照片修复模块**：解决偏黄、褪色、颗粒、模糊、人像不清、折痕划痕等质量问题；
- **纪念版生成模块**：把修复结果转换成相册、胶片、明信片、淡彩手绘等适合展示的纪念图；
- **VGG 自定义艺术风格模块**：提供更自由的高级个性化风格迁移；
- **高级算法功能**：支持课堂展示、参数实验和进一步精修。

系统达到了“修复可用、输出可展示、算法可解释”的目标，既能体现数字图像处理方法在实际照片修复中的作用，也能形成完整的课程项目演示闭环。
