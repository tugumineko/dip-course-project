# 家庭老照片修复与纪念化生成 — 项目设计稿

> 课程：ECNU SEI《数字图像处理_2026》（曹桂涛 教授）期末项目  
> 形态：单人 · tkinter 桌面程序 · 定制传统修复滤镜 + VGG 高级艺术风格扩展  
> 运行环境：conda `ai-service`（`E:\miniconda3\envs\ai-service\python.exe`，GPU 就绪）

---

## 1. 设计目标

本项目实现一个家庭老照片修复与纪念化生成系统。系统围绕三类输出目标设计：

1. **修复图**：改善发黄、褪色、灰蒙、颗粒、模糊、折痕划痕等质量问题；
2. **纪念模板图**：生成相册、胶片、明信片、淡彩手绘等适合家庭展示的结果；
3. **自定义艺术风格图**：使用 VGG19 Gatys 风格迁移支持用户上传风格图。

主使用流程：

```text
打开老照片 → 选择修复模式 → 查看前后对比 → 生成纪念版 → 保存或导出
```

## 2. 功能模块与目的

### 2.1 家庭老照片修复

| 模式 | 目的 | 需求来源 | 算法组合 |
| :--- | :--- | :--- | :--- |
| 温和修复 | 给大多数旧照提供自然改善 | 轻微发黄、灰蒙、颗粒 | 轻度 NLM、黄化校正、低强度 CLAHE、亮度 USM |
| 泛黄褪色修复 | 校正颜色老化 | 偏黄、偏棕、褪色 | LAB 限幅去黄、L 通道增强、轻微饱和恢复 |
| 黑白旧照增强 | 改善灰度层次和相纸感 | 黑白照层次弱、边缘不清 | 灰度去噪、CLAHE、S 曲线、暖纸色 |
| 扫描颗粒修复 | 减少颗粒并保护轮廓 | 扫描件背景颗粒、暗部脏点 | YCrCb 分离、色度强去噪、亮度弱去噪、边缘权重融合 |
| 人像清晰化 | 提升人物轮廓 | 家庭合影、人像略模糊 | 双边基础层、细节层、梯度 mask、亮度 gate |
| 折痕划痕辅助修复 | 修补局部破损 | 白线、裂痕、折痕 | top-hat / black-hat 自动掩膜、用户掩膜、inpaint |

### 2.2 纪念版生成

| 模板 | 目的 | 实现方式 |
| :--- | :--- | :--- |
| 复古相册风 | 生成适合家庭相册保存的暖色旧照 | 低饱和、暖色曲线、暗角、米色边框 |
| 暖色胶片风 | 生成生活照式怀旧质感 | S 曲线、暖高光、轻微颗粒、暗角 |
| 纪念明信片风 | 生成适合展示或赠送的卡片 | 米色画布、白边、投影、标题区域 |
| 淡彩手绘风 | 生成柔和绘画感纪念图 | 双边平滑、浅线稿、色彩融合 |

### 2.3 VGG 高级自定义艺术风格

| 功能 | 目的 | 实现方式 |
| :--- | :--- | :--- |
| 自定义艺术风格迁移 | 支持用户上传任意风格图 | 预训练 VGG19、内容特征、Gram 矩阵、L-BFGS 像素优化 |
| 暖调家庭风格参考 | 提供更贴合家庭纪念主题的示例风格 | `warm_family_memory_style.jpg` |

### 2.4 高级算法功能

用于课程展示、参数实验和精修调节，包含基础操作、增强、去噪、锐化、分割、形态学、图像恢复、频域处理、小波去噪等。

## 3. 新增代码模块

```text
代码/dip/family_restore.py      # 家庭老照片修复模式
代码/dip/commemorative.py       # 纪念版模板
代码/utils/make_family_style.py # 暖调家庭风格参考图生成脚本
```

### 3.1 `family_restore.py`

- `gentle_restore(img)`：温和修复；
- `restore_faded_yellow_photo(img)`：泛黄褪色修复；
- `restore_bw_photo(img)`：黑白旧照增强；
- `reduce_photo_grain(img)`：扫描颗粒修复；
- `portrait_clarity(img)`：人像清晰化；
- `scratch_assisted_inpaint(img, mask=None, auto=True)`：折痕划痕辅助修复；
- `scratch_mask_preview(img)`：自动划痕掩膜预览。

### 3.2 `commemorative.py`

- `vintage_album_style(img)`：复古相册风；
- `warm_film_style(img)`：暖色胶片风；
- `postcard_style(img)`：纪念明信片风；
- `soft_handpainted_style(img)`：淡彩手绘风。

## 4. GUI 结构

左侧功能区按使用顺序组织：

1. **家庭老照片修复**：六个定制修复模式 + 手动掩膜修复；
2. **纪念版生成**：四个传统模板 + VGG 自定义艺术风格；
3. **高级算法功能**：传统图像处理功能，用于展示和精修。

预览区显示原图和当前结果，参数区服务于高级算法功能，菜单提供打开、保存、导出对比图、撤销、重置和直方图查看。

## 5. 展示与评估

`代码/generate_gallery.py` 生成：

- `custom_*.jpg`：家庭老照片修复模式对比图；
- `memory_*.jpg`：纪念模板对比图；
- `vgg_家庭暖调风格迁移.jpg`：VGG 高级风格迁移示例；
- `metrics.json` 和 `metrics_chart.jpg`：合成样张量化指标。

评估采用双轨：

- 合成老照片：计算 PSNR / SSIM；
- 真实老照片：观察偏色、噪声、清晰度、自然度和纪念效果。

---

## 6. 边界说明

- 不训练深度学习修复模型；
- 不做上色；
- 自动划痕检测作为辅助功能，复杂破损仍支持用户掩膜；
- VGG 风格迁移用于高级自定义风格，主纪念模板使用传统图像处理实现。
