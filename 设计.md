# 家庭老照片修复与纪念化生成 — 项目设计稿

> 课程：ECNU SEI《数字图像处理_2026》（曹桂涛 教授）期末项目  
> 形态：单人 · tkinter 桌面程序 · 定制传统修复滤镜 + VGG 高级艺术风格扩展  
> 运行环境：conda `ai-service`（`E:\miniconda3\envs\ai-service\python.exe`，GPU 就绪）

---

## 1. 设计定位

项目从“算法工具箱”调整为“家庭老照片定制修复应用”。主流程不要求用户理解图像处理参数，而是按家庭照片场景提供固定处理方案：

```text
打开老照片 → 选择定制修复模式 → 生成家庭纪念模板 → 可选 VGG 自定义艺术风格 → 保存/导出
```

完整算法工具箱仍保留，用于课堂展示和精修调参，但不作为普通用户第一入口。

---

## 2. 核心功能结构

### 2.1 家庭老照片修复

| 模式 | 目标 | 算法组合 |
| :--- | :--- | :--- |
| 温和修复 | 大多数老照片默认方案 | 轻度 NLM、黄化校正、低强度 CLAHE、亮度 USM |
| 泛黄褪色修复 | 去掉过量黄化但保留暖色 | LAB 限幅去黄、L 通道增强、轻微饱和恢复 |
| 黑白旧照增强 | 增强灰度层次与旧相纸质感 | 灰度去噪、CLAHE、S 曲线、暖纸色 |
| 扫描颗粒修复 | 减少颗粒并保护边缘 | YCrCb 分离、色度强去噪、亮度弱去噪、边缘权重融合 |
| 人像清晰化 | 提升人物轮廓但不放大暗部噪声 | 双边基础层、细节层、梯度 mask、亮度 gate |
| 折痕划痕辅助修复 | 修补细线划痕和局部破损 | top-hat / black-hat 自动掩膜 + 用户掩膜 + inpaint |

### 2.2 纪念版生成

主流程使用传统图像处理模板，避免默认使用名画风格：

| 模板 | 处理方式 |
| :--- | :--- |
| 复古相册风 | 低饱和、暖色曲线、暗角、米色边框 |
| 暖色胶片风 | S 曲线、暖高光、轻微颗粒、轻暗角 |
| 纪念明信片风 | 米色画布、白边、投影、标题区域 |
| 淡彩手绘风 | 双边平滑、浅线稿、低强度色彩提升 |

### 2.3 VGG 高级自定义艺术风格

Gatys + VGG19 仍保留，但作为高级功能：

- 不训练模型；
- 支持用户上传任意风格图；
- 新增 `warm_family_memory_style.jpg` 作为更贴近家庭纪念的暖调风格参考；
- 不再把《星空》《神奈川冲浪里》作为家庭纪念主视觉。

## 5. 展示与评估

`代码/generate_gallery.py` 生成三类素材：

1. 定制修复对比图：`custom_*.jpg`
2. 纪念模板对比图：`memory_*.jpg`
3. 高级工具箱示例和指标图：`ch*.jpg`、`metrics.json`、`metrics_chart.jpg`

评估采用双轨：

- 合成老照片：PSNR / SSIM；
- 真实老照片：主观评价自然度、偏色、噪声、清晰度、纪念感。

说明：定制修复和纪念模板主动改变色彩与风格，不以 PSNR 最大为唯一目标。

## 7. 风险与边界

- 不做深度学习修复、超分和上色训练；
- 自动划痕检测只作为辅助，不承诺完全自动；
- VGG 风格迁移耗时较长，所以不作为默认纪念模板；
- 主纪念模板使用传统图像处理，保证速度和稳定性。
