# 家庭老照片修复与纪念化生成

华东师大《数字图像处理_2026》期末项目。

**做什么**：针对家庭老照片的黄化、褪色、颗粒、轻微模糊、人像不清、折痕划痕等问题，提供定制修复模式；修复后可生成复古相册、暖色胶片、纪念明信片、淡彩手绘等纪念版。VGG19 Gatys 风格迁移保留为高级自定义艺术风格功能。

## 主流程

1. 打开老照片。
2. 选择定制修复模式：温和修复、泛黄褪色修复、黑白旧照增强、扫描颗粒修复、人像清晰化、折痕划痕辅助修复。
3. 选择纪念版模板：复古相册风、暖色胶片风、纪念明信片风、淡彩手绘风。
4. 可选使用“自定义艺术风格迁移（VGG）”。
5. 保存结果或导出前后对比图。

## 功能概览

- **家庭老照片修复**：面向真实老照片场景的固定处理流水线，不要求用户理解算法参数。
- **纪念版生成**：主流程使用传统图像处理滤镜，速度快、风格更贴家庭纪念。
- **VGG 高级功能**：Gatys + 预训练 VGG19，支持上传自定义风格图，零训练。
- **高级工具箱**：保留增强、去噪、锐化、分割、形态学、恢复、频域、小波等经典功能，便于课程展示和精修。
- **评估与展示**：合成退化样张支持 PSNR / SSIM；真实老照片用于主观前后对比。

## 运行

必须用 conda `ai-service` 环境（含 GPU 版 torch）：

```powershell
E:\miniconda3\envs\ai-service\python.exe 代码\main.py
```

生成展示素材和幻灯片：

```powershell
E:\miniconda3\envs\ai-service\python.exe 代码\generate_gallery.py
E:\miniconda3\envs\ai-service\python.exe 代码\build_slides.py
```
