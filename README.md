# 老照片修复与风格化迁移纪念

华东师大《数字图像处理_2026》期末项目。

**做什么**：导入一张发黄、灰蒙、有颗粒噪声、轻微模糊或带划痕的老照片，通过“一键修复 + 针对性精修 + 艺术纪念版生成”的流程完成修复与纪念化输出。

## 使用流程

1. 打开老照片。
2. 使用“一键温和修复”得到基础结果。
3. 根据问题继续精修：去黄偏色、提亮增强、去颗粒噪声、增强清晰度、划痕/折痕修复。
4. 可选生成艺术纪念版。
5. 保存结果或导出前后对比图。

## 功能概览

- **修复向导**：面向普通用户，按老照片常见问题组织入口。
- **高级工具箱**：保留增强、去噪、锐化、分割、形态学、恢复、频域、小波等经典图像处理功能，便于调参和课堂展示。
- **艺术纪念版**：Gatys 风格迁移，使用预训练 VGG19，零训练、支持自定义风格图。
- **评估与展示**：合成退化样张支持 PSNR / SSIM；真实老照片用于主观前后对比。

## 运行

必须用 conda `ai-service` 环境（含 GPU 版 torch）：

```powershell
E:\miniconda3\envs\ai-service\python.exe 代码\main.py
```

工具脚本可单独跑：

```powershell
# 合成一张“老照片”
E:\miniconda3\envs\ai-service\python.exe 代码\utils\degrade.py 素材\干净原图\test.jpg 素材\合成老照片\test_old.jpg --mask 素材\合成老照片\test_mask.png

# 算指标
E:\miniconda3\envs\ai-service\python.exe 代码\utils\metrics.py 素材\干净原图\test.jpg 素材\合成老照片\test_old.jpg
```
