# 演示视频

将录制好的系统演示视频放在本目录，并命名为：

```text
demo.mp4
```

然后在项目根目录运行：

```powershell
E:\miniconda3\envs\ai-service\python.exe 代码\build_slides.py
```

脚本会把 `demo.mp4` 转成 base64 并直接嵌入 `演示文稿.html`，最终 HTML 可以像 PPT 一样单文件携带和播放，不再依赖外部视频文件。

建议录制规格：

- 时长：70–90 秒
- 画面：横屏 16:9
- 格式：MP4 / H.264，尽量压缩到 100 MB 以内，避免 HTML 过大
- 内容顺序：打开 App → 导入老照片 → 温和修复 / 泛黄褪色 / 人像清晰化或扫描颗粒 → 纪念明信片风 → 复古相册风或暖色胶片风 → VGG 面板和预生成结果 → 保存或导出对比图
